# vue-tariffs
Простой проект на Vue, предлагающий пользователю выбрать один из тарифов интернета

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes tarif
```
npm run lint
```
